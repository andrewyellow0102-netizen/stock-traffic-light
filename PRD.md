# 股票紅綠燈系統 — PRD

## 1. 概述

**名稱：** Stock Traffic Light（股票紅綠燈）  
**類型：** Web 應用（台股分析工具）  
**核心功能：** 
1. 輸入股號，查詢目前紅綠燈（買進/觀望/賣出）
2. 根據燈號篩選股票，找出目前最佳投資標的

**目標使用者：** 投資台股的散戶，想快速判斷個股進場時機

---

## 2. 紅綠燈判斷邏輯

### 燈號演算法（技術分析版）

| 燈號 | 條件 |
|------|------|
| 🟢 **綠燈（買進）** | 滿足以下任一：①股價低於月線 10% 以上（超跌）、②RSI < 30（超賣）、③KD 黃金交叉且 K < 30 |
| 🟡 **黃燈（觀望）** | 不滿足綠也不滿足紅，沒有明確方向 |
| 🔴 **紅燈（賣出）** | 滿足以下任一：①股價高於月線 15% 以上（超漲）、②RSI > 70（超買）、③KD 死亡交叉且 K > 70 |

### 資料來源
- 即時股價：TWSE OpenAPI (`https://mis.twse.com.tw/zh/`)
- 日線資料：用於計算均線、RSI、KD（從 TWSE 或過往快取）
- **MVP 階段**：使用 yfinance 取得歷史資料計算指標（未來再串 TWSE）

---

## 3. 功能規格

### 3.1 股號查詢
- 輸入框：接受股號（如 `2330`、`0050`）
- 顯示：股票名稱 + 現價 + 燈號 + 簡單理由
- 技術指標顯示：均線位置、RSI、KD 數值

### 3.2 燈號股票搜尋
- 按鈕切換：只看「🟢 綠燈股票」/ 「🔴 紅燈股票」/ 「全部」
- 熱門股票列表（預設清單：0050, 0056, 2330, 2317, 2884...）
- 可自訂股票清單（localStorage）
- 點任一股票可快速看燈號詳情

### 3.3 燈號詳情面板
- 點擊股票展開：顯示完整技術分析
- 圖形化：均線、K 線（用簡單的文字 or Chart.js）
- 買賣理由文字說明
