## 兩套策略並行的系統

### 策略 A：短線價差操作（逆勢型態交易）

利用 K 線型態在相對**高點賣出、低點買進**，捕捉反轉轉折點。

#### 常見反轉型態（高檔賣出）

| 型態 | 辨識方法 | 訊號意義 |
|------|---------|---------|
| **雙星變盤** | 高檔連續兩根變盤 K 棒 | 高點賣出警訊 |
| **高檔母子** | 母嬰孕育組合 | 波段高點逃命訊號 |
| **孤島夜星** | 跳空後連續反轉 | 強烈空頭確認 |
| **高檔群星變盤** | 多根小型 K 聚集後反轉 | 多轉空關鍵訊號 |
| **三根 K 棒判斷** | 連續三根 K 棒方向研判趨勢 | 快速確認轉向 |
| **5 分 K 主力蹤跡** | 透過分 K 觀察主力何時動手 | 精準進場時機 |

#### 關鍵心法
- **耐心等待完整訊號**才進場，不提前進場
- 進場訊號未完整就忍不住出手 → 常見心魔，需要刻意練習等待節奏
- 停損紀律果斷，絕不攤平

---

### 策略 B：長線持有（順勢而為）

在明確多頭趨勢中持有至趨勢結束，適合波段操作。

#### 趨勢確認工具

| 工具 | 用途 | 範例 |
|------|------|------|
| **均線多頭排列** | 確認長線上漲趨勢 | MA5 > MA20 > MA60 |
| **MA 黃金交叉** | 波段買進確認 | MA20 突破 MA60 |
| **N 字底突破** | 趨勢起漲點確認 | 突破前高且成交量放大 |
| **W 底 / 頭肩底** | 底部反轉型態 | 兩次測試支撐不破後拉升 |
| **V 轉型態** | 急速反轉捕捉 | 急跌後迅速收復失土 |
| **飄旗型態** | 中段整理後延續 | 旗型整理後持續順向 |
| **財報 + 技術面共振** | 找出真正會漲的股票 | 基本面佳且型態佳 |

#### 多頭結束警訊
- 跌破均線（但要分辨是洗盤還是真正轉弱）
- 高檔震盪時判斷：均線支撐是否有效
- KD 高檔死亡交叉（K > 70 時死叉）

## K 線型態速查表

### 底部反轉型態（買進參考）

| 型態 | 圖形特徵 | 確認條件 |
|------|---------|---------|
| W 底 | 兩次測試相近低點後拉升 | 第二底不破前低，突破頸線 |
| 頭肩底 | 左肩、頭部、右肩 | 突破頸線且成交量增加 |
| N 字底 | 一底高一底更高 | 突破前高 |
| V 轉 | 急速下跌後迅速收復 | 長下影線 + 大陽線 |

### 頂部反轉型態（賣出警訊）

| 型態 | 圖形特徵 | 確認條件 |
|------|---------|---------|
| M 頭 | 兩次測試相近 高點後下跌 | 跌破頸線 |
| 頭肩頂 | 左肩、頭部、右肩 | 跌破頸線 |
| 雙星變盤 | 高檔連續變盤 K 棒 | 第二根確認後賣出 |
| 高檔母子 | 小實體在 大實體下方 | 母線涵蓋子線 |
| 孤島夜星 | 跳空上漲後連續下跌 | 封閉缺口 |
| 島狀反轉 | 竭盡缺口後島型反轉 | 成交量放大確認 |

## 應用於股票紅綠燈系統

現有系統已具備：RSI、KD、MACD、MA 均線、買賣燈號、產業分類

### 可強化項目

```
1.【成交量缺口篩選】
   條件：今日成交量 > 近20日均量 2倍 → 可能是真突破
   
2.【均線偏離度警示】
   條件：股價偏離 MA20 超過 ±15% → 留意回調風險
   
3.【K線型態標註】（需人工收盤後標註）
   當日是否出現「雙星」「母子」「孤島」「N字」等型態
   
4.【多空趨勢判斷增強】
   股價在 MA60 之上 → 多頭格局（長線）
   股價在 MA20 之上 → 中線多頭
   跌破 MA20 → 短線偏空
```

## 每月/每日實作清單

### 每晚（10 分鐘）
- [ ] 對照當日強勢股，檢視是否有老師教的型態出現
- [ ] 符合型態的股票加入自選，隔日觀察
- [ ] 記錄「為什麼我此時進場」的原因

### 每筆交易
- [ ] 進場前：確認訊號完整（不只是均線黃金交叉，要看 K 線型態）
- [ ] 停損設定：-7% 果斷退出，不攤平
- [ ] 賣出後：回顧這次操作是否符合型態紀律

### 每月檢視
- [ ] 統計哪些型態勝率最高
- [ ] 專注自己擅長的 2～3 種型態，反覆練習
- [ ] 淘汰成效不好的型態

## 老師的三大紀律

1. **訊號未完整，絕對不進場** — 提前進場是最大虧損來源
2. **停損果斷，絕對不攤平** — 判斷錯誤時最快止損
3. **只操作自己看得懂的型態** — 貪多會乱，專注才能精準
